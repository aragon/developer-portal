Actions are represented by solidity struct and executed by the DAO's `execute` function, resulting in an external call.


```graphql
type Action {
  dao: Dao!
  data: Bytes!
  execResult: Bytes
  id: ID!
  proposal: IProposal!
  to: Bytes!
  value: BigInt!
}
```
