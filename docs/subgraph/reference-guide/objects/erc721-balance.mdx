The NFT balances of the DAO. Including all Token IDs held by the DAO per [ERC721 Token Smart Contract](https://eips.ethereum.org/EIPS/eip-721).


```graphql
type ERC721Balance implements TokenBalance {
  dao: Dao!
  id: ID!
  lastUpdated: BigInt!
  token: ERC721Contract!
  tokenIds: [BigInt!]!
}
```
