Wrapped [ERC20 Token Smart Contract](https://eips.ethereum.org/EIPS/eip-20). Used for token voting.

```graphql
type ERC20WrapperContract implements Token {
  decimals: Int!
  id: ID!
  name: String
  symbol: String
  underlyingToken: ERC20Contract!
}
```
