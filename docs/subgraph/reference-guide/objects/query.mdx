> 
> The block at which the query should be executed. Can either be a `&#x007B; hash: Bytes &#x007D;` value containing a block hash, a `&#x007B; number: Int &#x007D;` containing the block number, or a `&#x007B; number_gte: Int &#x007D;` containing the minimum block number.
